# Shelldex Project YAML Schema

Each project is a single `.yml` file in `src/data/projects/`.

---

## Required Fields

| Field         | Type       | Description                                                   |
|---------------|------------|---------------------------------------------------------------|
| `name`        | `string`   | Display name of the project                                   |
| `slug`        | `string`   | URL-safe identifier (used in `/projects/{slug}`)              |
| `tagline`     | `string`   | One-line pitch (shown on cards)                               |
| `description` | `string`   | Longer description (1–3 sentences)                            |
| `language`    | `string`   | Primary language — one of: `typescript`, `rust`, `python`, `go`, `c`, `bash`, `zig`, `multi` |
| `category`    | `string[]` | One or more categories (e.g. `[security, lightweight]`)       |
| `status`      | `string`   | One of: `active`, `experimental`, `archived`                  |
| `highlight`   | `string`   | Key selling point or notable metric                           |
| `emoji`       | `string`   | Single emoji representing the project                         |
| `added`       | `string`   | ISO date when the project was added (`YYYY-MM-DD`)            |

## Optional Fields

| Field     | Type     | Description                                    |
|-----------|----------|------------------------------------------------|
| `github`  | `string` | GitHub repo in `owner/repo` format             |
| `website` | `string` | Project homepage URL                           |
| `tier`    | `number` | Tier ranking: `1`, `2`, or `3`                 |

## Enriched Fields — `github_data`

Added automatically by `npm run enrich`. **Do not edit manually.**

```yaml
github_data:
  stars: 228000           # Current stargazers count
  forks: 14200            # Fork count
  open_issues: 842        # Open issue count
  license: MIT            # SPDX license identifier (or null)
  topics:                 # GitHub repo topics
    - ai
    - agent
    - self-hosted
  last_commit: "2026-02-27T18:30:00Z"          # pushed_at timestamp
  latest_release: "v4.2.1"                      # Tag name of latest release (or null)
  latest_release_date: "2026-02-20T12:00:00Z"   # Release publish date (or null)
  star_growth_7d: 1200                          # Stars gained in last 7 days (or null if no history)
  last_enriched: "2026-02-28T10:00:00Z"         # ISO timestamp of last enrichment run
```

| Field                 | Type              | Description                                              |
|-----------------------|-------------------|----------------------------------------------------------|
| `stars`               | `number`          | Current stargazer count from GitHub API                  |
| `forks`               | `number`          | Fork count                                               |
| `open_issues`         | `number`          | Open issues count                                        |
| `license`             | `string \| null`  | SPDX license ID (e.g. `MIT`, `Apache-2.0`)              |
| `topics`              | `string[]`        | GitHub repository topics                                 |
| `last_commit`         | `string`          | ISO timestamp of the repo's last push                    |
| `latest_release`      | `string \| null`  | Tag name of the latest GitHub release                    |
| `latest_release_date` | `string \| null`  | ISO timestamp of the latest release                      |
| `star_growth_7d`      | `number \| null`  | Stars gained in the last 7 days (null on first run)      |
| `last_enriched`       | `string`          | ISO timestamp of when enrichment last ran                |

## Leaderboard JSON — `src/data/generated/leaderboard.json`

Generated by the enrichment script. Contains pre-computed rankings and health indicators.

```json
{
  "generated": "2026-02-28T10:00:00Z",
  "count": 27,
  "entries": [
    {
      "rank": 1,
      "name": "OpenClaw",
      "slug": "openclaw",
      "emoji": "🦞",
      "language": "typescript",
      "github": "openclaw/openclaw",
      "stars": 228000,
      "forks": 14200,
      "open_issues": 842,
      "star_growth_7d": 1200,
      "latest_release": "v4.2.1",
      "latest_release_date": "2026-02-20T12:00:00Z",
      "last_commit": "2026-02-27T18:30:00Z",
      "license": "MIT",
      "health": {
        "issues_stars_ratio": 0.0037,
        "days_since_last_release": 8,
        "days_since_last_commit": 1
      }
    }
  ]
}
```

### Health Indicators

| Indicator              | Meaning                                                              |
|------------------------|----------------------------------------------------------------------|
| `issues_stars_ratio`   | `open_issues / stars` — lower is healthier (green < 0.01, yellow < 0.05, red >= 0.05) |
| `days_since_last_release` | Days since the latest GitHub release (null if no releases)        |
| `days_since_last_commit`  | Days since the repo was last pushed to                            |

## Supporting Files

| File                        | Purpose                                                    |
|-----------------------------|------------------------------------------------------------|
| `.enrichment-history.json`  | Star-count snapshots by date (for 7-day growth calc). Auto-managed, 30-day rolling window. |
| `.stars-cache.json`         | Build-time star cache used by `src/lib/projects.ts` (1-hour TTL). |

## Running Enrichment

```bash
# Requires a GitHub personal access token for higher rate limits
GITHUB_TOKEN=ghp_... npm run enrich

# Or without token (60 req/hr limit)
npm run enrich
```
